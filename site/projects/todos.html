To do list - The Game

I've been wanting to do a game with Godot for a good while now, and this is what I started with. The idea is to make something similar to Habitica : a game where you complete daily habits and tasks to earn coins, used to customize your character.

I have already hit a few roadblocks : doing a mostly UI game (I don't know shit about UI design), in pixel art (I barely ever did any pixel art), in an engine so different from anything I've ever used (what do you mean there's no DateTime in gdscript??). But I've been having fun so far, and that's what counts!
